An assistant must work on voice signals as well as convert the voice signals into text and vice versa. Python provide a library called pyttsx3. pyttsx3 is a text-to-speech conversion library in Python. Unlike alternative libraries, it works offline and is compatible with both Python 2 and 3. Very first this library is imported into code. Later a Speech recognition module as sris imported. The prime use of this module is to recognize speech and convert them into text. And more libraries and modules are implemented which are given below. ● Datetime (provides date-time functionality).

● Sys (The sys module provides information about constants, functions and methods of the Python interpreter). ● Wikipedia (assist in wikipedia search).

● Os(use in file handling).

● Webbrowser(directly access default web browser).

● Random (implements randint function).

● Smtplib.

● Pyjokes

To invoke pyttsx3 module a variable with name engine is defined and sapi5 is a default command which invokes pysttx3 module is pass. furthersapi5 helps to select type of voice with getproperty() and setproperty() method in form of a API. Since assistant need to speak a function speak() is defined with argument audio being pass to it. runandwait() is in-built function that enables our speak function to actually convert string to speech.

Basic function should be that it must tell date and time as well as wish users intelligently.To implement date and time datetime module is imported. datetime has functions to implement date and time attributes. such as datetme.now( ).hour. In below we have created a wishMe() function. The function uses a variable hour to implement the functions. And if-else ladder is used to wish according to time and day expressed in hours. After enabling program to converting string(s) to voice signal(s). It is important for Jarvis to take commands from user which means that program should be in co-ordination with an output devices such as microphone, so that it Jarvis can take input. speechrecoginition have in built functions that accepts input in form of voice signals. takecommand() implements speechrecoginition. In the function sr.recognizer() recognizes audio and microphone() is considered source. Further r.pause_threshold() decides for much time program must wait to accept input. audio argument that was passed in speak function is now assigned with listen() function that accepts input. Though program seems to be flawless yet is prone to errors to avoid this we used try method that has query assigned with recognize_google() method which use google cloud to realize language. Here we had used Indian English. An exception function is also in action to appeal user to retry if any error occurs. Here our process of Natural language processing (NPL) had been completed. In main method query is assigned with lower() method to display command in lower case
