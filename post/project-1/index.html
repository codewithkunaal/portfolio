Project 1: Fake News identifier - using BI-LSTM

August 4, 2022

Project : problem statement for the project was to classify a news into category of whether it’s fake or true and then further identify if we can track or Categories such news sources into legit or biased and to do that we need a model that will learn patterns from data and the data was taken from kaggle we had 2 data sets IEEE and ISOT which combined had over 70k records

During the literature survey we concluded that traditional machine learning methods were used but results weren’t promising so we wanted to go with more modern approach ie neural networks previously BERT and naive bayes were used but the accuracy of both models was low and naive bayes can’t correlate the features together so it’s kind of irreverent to find prominent sources so when we had datasets we merged them together and perform EDA which involved removing stop words, stemming(base form of word) and lemmitation (grammer) and padding (same length for each sentence) then machine don’t learn from the words we needed this text in form of numbers so we had to tokenize them which was done with word embedding (word2vec) and for the model buildings part we went with neural network RNN at first we used Lstm

In lstm it has only one layer that take tokens as input learns pattern and gives output and same layer is fed all the other words in different intervals but problem with lstm was that it cannot remember the significance of words that come at beginning of sentence so we needed new ways to catch this information thats why we use bi directional lstm in bi-lstm there is one more layer right below the rnn layer which traverse the sentence backwards and learn pattern from end to begining of sentence

After the model building we did an accuracy check and got… accuracy which was quite low so then we tuned hyper-parameters which resulted in 98…accuracy so our model is capable of classifying news as true of fake with 98 accuracy

I was actively involved in literature survey and model building and hyper parameter tuning
